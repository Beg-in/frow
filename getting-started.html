Getting Started With Frow

Initial page setup

1. Install Frow

Bower:

$ bower install frow

Or, npm:

$ npm install frow

Or, if you must, download and use the precompiled CSS:

<!DOCTYPE html>
<html>
    <head>
        <link rel="stylesheet" href="frow/css/frow.min.css">
        ...
    </head>
    ...
</html>
2. Include HTML5 DOCTYPE
<!DOCTYPE html>
<html>
...
</html>
3. Add mobile viewport meta tag

Recommended:

<!DOCTYPE html>
<html>
    <head>
        <meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1, user-scalable=no">
        ...
    </head>
    ...
</html>

Or, to allow mobile zooming:

<!DOCTYPE html>
<html>
    <head>
        <meta name="viewport" content="width=device-width, initial-scale=1">
        ...
    </head>
    ...
</html>
4. Import Frow into your Sass file
// @import 'variables';
// Tip: Place your variables file first to overrule any of Frow's defaults
@import 'frow';

Don't forget to add vendor prefixes and minify after converting to CSS!
