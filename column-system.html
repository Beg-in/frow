The Column System

Columns Divide Up the Rows

After declaring a ".frow", or flex row, the inside content of the div becomes CSS display: flex. With this you can use columns to break content apart via fractions. For example .col-md-1-3 creates a column that is one third of the row. With the default Frow settings, you can use any fraction from halfs to twelths.

Flex Is Love, Flex Is Life

So after that basic gridwork, lets see where Frow stands out. Bootstrap could already handle most of the previous examples, but now lets dive into the deep end!

If the fractions add up to less than a whole, Frow makes them centered and nice. Or use simple rules to change the behaviour.

class="frow" // Centers columns horizontally by default

.col-md-6-10

class="frow row-start"

.col-md-6-10

class="frow row-end"

.col-md-6-10

CSS Flex even allows for elements to be centered horizontally AND vertically! With Frow, one class and you're done.

class="frow centered"

.col-md-1-3

Gutters Optional

Frow is designed to work both as your grid system as well as on design styles within the grid. For the overall grid system, gutters between the columns are probably a good idea. Just add the .gutters class to the .frow and you're done. With default settings, the gutters are 30 pixels wide.

class="frow gutters"

.col-md-1-4

.col-md-2-4

.col-md-1-4

Vertical Columns Work Too

Frow can layout elements in vertical columns! Just add .column-center, .column-start, or .column-end.

class="frow column-center"
