Frow

Flex Row CSS Grid System

What is Frow?

Frow (flex row) is a grid system for CSS utilizes and is 100% based on, you guessed it, the CSS display property: flex.

The purpose of Frow is to be a replacement in some regards to Bootstrap's grid system, however Frow does not plan to grow to the whole scope of Bootstrap's features. For example: No Javascript plugins.

How to use it

Column System

After declaring a ".frow", the inside content of the div becomes CSS display: flex. With this you can use columns to break content apart via fractions. For example .col-md-1-3 creates a column that is one third of the row. With the default Frow settings, you can do any fraction from halfs to twelths.

Flex is love, Flex is life

So after that basic gridwork, lets see where Frow stands out. Bootstrap could already handle most of the previous examples, but now lets dive into the deep end!

If the fractions don't add up to one, Frow makes them centered and nice. Or use simple rules to change the behaviour.

class="frow"

.col-md-6-10 (centered by default)

class="frow row-start"

.col-md-6-10 (left-align)

class="frow row-end"

.col-md-6-10 (right-align)

CSS Flex even allows for elements to be centered horizontally AND vertically! With Frow, one class and you're done.

class="frow centered"

.col-md-1-3 (inside .frow.centered)

Gutters optional

Frow is designed to work both as your grid system as well as on design styles within. For the overall grid system, gutters between the columns are probably a good idea. Just add the .gutters class to the .frow and you're done. With default settings, the gutters are 30 pixels wide.

class="frow gutters"

.col-md-1-4

.col-md-2-4

.col-md-1-4
