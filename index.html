An Introduction to Frow

What Frow is:

Frow (flex row) is a grid system for CSS utilizes and is 100% based on, you guessed it, the CSS display property: flex.

If you know the CSS property display:flex, then you are going to love Frow. If you don't know it yet, then you are going to love the CSS property display:flex, and thus, Frow.

What Frow isn't:

Frow isn't a "full" Bootstrap replacement. Frow is a super cool grid system with some very minor styling tools. There are no JavaScript plugins. There are no icons. There are only a limited amount of components.

Frow isn't finished. It is brand new and receiving updates all the time. Until version "1.0.0" is hit, consider this a beta.
