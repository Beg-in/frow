Goodies

Little extras to make your day

Clickable/Not-Clickable

Two simple classes to change which cursor appears on an element. Clickable's rule is also automatically applied to any elements with the attribute of onclick or Angular's ng-click.

// These would all make the cursor change to a pointer on hover
<div class="clickable"></div>
<div onclick="function()"></div>
<div ng-click="function()"></div>

// This would force the cursor to not change to a pointer
<a href="http://usefrow.com" class="not-clickable"></a>
Full height rules

These two classes simply add a quick height:100% or min-height:100% rule to the elements. Comes in super handy in certain cases, especially when wanting to do full height webpages, where you have to make sure all elements are height:100% from <html> and <body> down.

// A full height div
<html class="height-100">
    <body class="height-100">
        <div class="height-100" style="background-color:#03A9F4;color:#fff;">
            Cool!
        </div>
    </body>
</html>

// Webpage that
<html class="height-100">
    <body class="height-100">
        <div class="frow centered">
            Cool!
        </div>
    </body>
</html>
Clearfix

As a Frow enthusiast, I strongly encourage you to figure out a way to do EVERYTHING with display:flex, but for those times when you absolutely have to use a CSS float, I've included a clearfix class.

<div style="float:right;">Eww.</div>
<div class="clearfix"></div> // Just add this line

*This website is 100% organic and float-free.
